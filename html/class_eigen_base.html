Detailed Description

template<int dim>
class EigenBase< dim >

This class provides eigenvalue calculations foundations.

This class is the base class of eigenvalue calculations. It will operate the iteration at a high level without touching specifics in the equations. Alternatively, it interfaces with MGBase objects iteratively.
