Detailed Description

template<int dim>
class BartDriver< dim >

This class operate BART. The functionalities include: (1) Build necessary components for BART, such as AQ data, mesh, equations etc. For This functionality, please see documentation for BartDriver<dim>::build_basis. (2) Setting up system. (3) Reporting system on screen. (4) Operation calculations at highest level: instantiating Iterations and perform iterations. (5) Output results for visualization.

This function outputs results. The main functionality is to provide output files that can be read by Paraview or Visit.
